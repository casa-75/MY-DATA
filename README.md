# 📁 Dokumen Saya

Repository ini berisi kumpulan foto, dokumen, dan file lainnya yang dapat diunduh secara bebas.

## 📂 Struktur Folder

| Folder | Isi |
|--------|-----|
| `foto/` | Kumpulan foto dan gambar |
| `dokumen/` | File PDF (laporan, proposal, dll) |
| `lainnya/` | File lain-lain |

## 📥 Cara Download

1. Buka folder yang sesuai dengan file yang kamu cari
2. Klik nama file yang diinginkan
3. Klik tombol **Download raw file** (ikon panah ke bawah) di pojok kanan atas halaman preview
